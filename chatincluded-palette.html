How to Use It

Website

  • Slate Deep as the page background
  • Sun Core for all CTAs and key links
  • Slate Mid for cards and feature blocks
  • Cream for headlines, Slate Mist for body
  • Sun Glow as a subtle radial glow behind hero

Twitter/X

  • Sun Core as your profile accent color
  • Header image: Slate Deep background + Sun Core wordmark
  • Graphics: dark base with yellow highlights
  • Avoid white backgrounds — stay in the dark palette
  • Sun Glow for quote graphics and announcements

Do

  • Let yellow breathe — use it sparingly for impact
  • Pair Slate Deep + Cream for readable dark mode text
  • Use Sun Warm on hover, never as default
  • Keep backgrounds dark to feel native to gaming

Don't

  • Don't put yellow text on cream — low contrast
  • Don't use bright white — use Cream instead
  • Don't add other accent colors — keep it two-tone
  • Don't use gradients on buttons — flat yellow only
